网络摄像头

开启摄像头(WebRTC)

描述

开启摄像头，并向WebRTC服务器注册

配置

id: 字符串 必填项 配置该摄像头的id，用于之后和video控件的配对

server ip： IP地址 默认值127.0.0.1 必填项 WebRTC服务器的地址

输入

switch: 布尔值 摄像头的开关信号。开表示想服务器注册，关表示注销

输出

status: 布尔值 如果注册成功，输出true，如果注销成功，输出false
