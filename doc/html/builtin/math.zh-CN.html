数学库

加法

描述

out = in1 + in2

输入

in1: 数值类型

in2: 数值类型

输出

out: 数值类型 out = in1 + in2

样例

example

in1:2.5

in2:-1

out输出1.5： 2.5 + (-1) = 1.5

减法

描述

out = in1 - in2

输入

in1: 数值类型 被减数

in2: 数值类型 减数

输出

out: 数值类型 out = in1 - in2

样例

example

in1:-2

in2:1.5

out输出-3.5： (-2) - 1.5 = (-3.5)

乘法

描述

out = in1 x in2

输入

in1: 数值类型

in2: 数值类型

输出

out: 数值类型 out = in1 x in2

样例

example

in1:-2

in2:-1.1

out输出2.2： (-2) x (-1.1) = 2.2

除法

描述

out = in1 / in2

输入

in1: 数值类型 被除数

in2: 数值类型 除数

输出

out: 数值类型 out = in1 / in2

example
